Design philosophy

  • Safety in code — config can disarm, never widen
  • Fail-closed money path — ambiguity refuses submit
  • Fail-open intelligence — LLM never blocks sanctioned trades on error

What AI can do

  • Observe health & positions
  • Relay operator commands
  • Veto with skip (when enabled)

What AI cannot do

  • Set size, side, leverage
  • Bypass any gate
  • Call an exchange SDK

All servers bind 127.0.0.1 only. Sole public surface: Tailscale Funnel → /webhook.
